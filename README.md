# Influence-Warfare---Neural-State-Reorganisation
Supplementary analysis and figures supporting the exploratory spectral analyses presented in "Influence Warfare Beyond Narratives: Cognitive State Manipulation"

This repository contains the supplementary figures and processed CSV outputs supporting the exploratory spectral analyses presented in the accompanying paper, Influence Warfare Beyond Narratives: Cognitive State Manipulation. The material is provided to maximise transparency and reproducibility while maintaining the readability and focus of the manuscript.

The repository includes the complete set of supplementary spectral visualisations and their corresponding processed data outputs referenced throughout the paper. Raw EEG datasets and analysis scripts are not included in this repository.
