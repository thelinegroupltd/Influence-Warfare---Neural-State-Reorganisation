# Differential Entropy and Lempel–Ziv Complexity Difference Analysis

## Isolating the Effect of Neuromodulation

This directory contains differential analyses of **Shannon Entropy** and **Lempel–Ziv Complexity (LZ76)** generated by subtracting the baseline (eyes closed) condition from the modulation session.

Rather than comparing multiple time-series independently, these plots directly visualise the magnitude and direction of change induced by the experimental condition.

## Difference Metrics

The analyses were computed as:

* **ΔH = H<sub>modulation</sub> − H<sub>baseline</sub>**
* **ΔLZ = LZ<sub>modulation</sub> − LZ<sub>baseline</sub>**

Positive values indicate increases relative to baseline, while negative values indicate reductions.

Differential entropy highlights where the informational characteristics of the EEG signal became more or less variable over time. Differential Lempel–Ziv complexity captures corresponding changes in the emergence of novel temporal patterns, providing a complementary measure of algorithmic complexity.

## Why Differential Analysis?

Absolute entropy and complexity traces require visual comparison between multiple recordings, making subtle modulation effects difficult to identify.

The differential approach removes the shared baseline dynamics and isolates only those changes attributable to the experimental condition. Regions exhibiting large positive or negative deviations therefore immediately identify periods during which neuromodulation altered cortical information dynamics.

## Systems Interpretation

Within the broader analytical framework of this repository, these plots emphasise **state transitions rather than absolute state**.

Rather than asking whether entropy or complexity increased or decreased overall, the differential analysis identifies **when** and **where** the brain departed from its baseline operating regime.

Together with the coherence, temporal persistence and state-space analyses, these figures support the interpretation that neuromodulation is best understood as a dynamic reorganisation of cortical state rather than a simple change in oscillatory power or signal amplitude.

This analysis provides a compact visual summary of modulation-induced changes and serves as a complementary representation of the absolute entropy and complexity measurements presented elsewhere in the repository.
